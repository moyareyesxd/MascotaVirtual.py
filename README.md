🐾 Waffle - Tu Mascota Virtual en Python
¡Bienvenido a Waffle, un juego clásico de mascota virtual basado en consola! Este proyecto es un simulador simple e interactivo escrito en Python donde tendrás que cuidar a Waffle, gestionando su energía y su felicidad para mantenerlo saludable.

🚀 Características
El juego funciona mediante un bucle en la terminal donde puedes elegir diferentes acciones para interactuar con Waffle:

🍎 Alimentar: Aumenta la energía (+20) y mejora un poco la felicidad (+1).
⚽ Jugar: Waffle se divierte mucho (+20 de felicidad), pero consume su energía (-15).
📊 Ver estado de salud: Revisa los niveles actuales de energía y felicidad.
💤 No hacer nada: El tiempo pasa, lo que aburre a Waffle (-10 de felicidad) y le quita un poco de energía (-5).
⚠️ Cuidado: Si la energía de Waffle llega a 0, ¡el juego terminará!

🛠️ Requisitos
Solo necesitas tener instalado Python 3.x en tu sistema. No requiere ninguna librería externa.

💻 Cómo ejecutar el proyecto
Clona este repositorio en tu máquina local: git clone https://github.com/TU_USUARIO/waffle-mascota-virtual.git Navega al directorio del proyecto:
Bash cd waffle-mascota-virtual Ejecuta el script:

Bash python mascotaVirtual.py
